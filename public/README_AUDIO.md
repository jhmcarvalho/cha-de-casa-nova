# Como adicionar a música "Hedwig's Theme"

Para que a trilha sonora funcione no site, você precisa adicionar o arquivo de áudio nesta pasta (`public`).

## Passos:

1. **Baixe a música "Hedwig's Theme"** do Harry Potter (formato MP3, OGG ou WAV)

2. **Renomeie o arquivo para:** `hedwigs-theme.mp3`

3. **Coloque o arquivo nesta pasta:** `public/hedwigs-theme.mp3`

4. **Pronto!** A música tocará automaticamente quando o site for aberto (ou após o usuário interagir com a página, dependendo das políticas do navegador)

## Alternativas de formato:

Se você tiver em outro formato, pode renomear para:
- `hedwigs-theme.ogg`
- `hedwigs-theme.wav`

E atualizar o código em `app/page.tsx` na linha onde está `/hedwigs-theme.mp3` para usar a extensão correta.

## Nota sobre direitos autorais:

Certifique-se de ter os direitos para usar a música ou use uma versão royalty-free/criative commons.
